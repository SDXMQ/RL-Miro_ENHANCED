# RL-Miro Enhanced (강화학습 미로 해결 시뮬레이터)

RL-Miro Enhanced는 Pygame 기반 GUI를 통해 에이전트가 미로를 스스로 학습하고 해결해 나가는 과정을 시각화하여 관찰할 수 있는 **강화학습 교육용 시뮬레이션 프로그램**입니다. 

---

## 🌟 주요 기능

1. **다양한 강화학습 알고리즘 지원**:
   * **Q-Learning**: 이상적인 최단 경로를 탐색하는 오프폴리시(Off-policy) 학습.
   * **SARSA**: 실제 행동을 타겟으로 삼아 안전지향적 경로를 선호하는 온폴리시(On-policy) 학습.
   * **Double Q-Learning**: 두 개의 Q-테이블을 교차 사용하여 Q값 과대평가 문제를 억제하는 신중한 학습.
2. **동적 미로 크기 조절 (5×5 ~ 30×30)**:
   * 슬라이더를 통해 격자 크기를 자유롭게 변경할 수 있습니다.
   * 미로 크기가 촘촘해지거나 커져도 UI(에이전트 원, 특수 타일, 라인 굵기)가 비율에 맞게 적응형으로 스케일링됩니다.
3. **미로 에디터 및 DFS 랜덤 미로 생성기**:
   * **편집 모드**: 마우스 클릭 및 드래그 브러시로 벽, 길, 함정, 보너스, 시작점, 목적지를 실시간 배치합니다. (우클릭은 지우기)
   * **🎲 랜덤 미로 생성**: DFS(깊이 우선 탐색) 알고리즘을 사용해 막힘없는 고유 미로를 원클릭으로 무작위 생성합니다.
4. **특수 타일 규칙**:
   * **함정 (Trap - 빨간 X)**: 대시보드 토글 옵션(ON/OFF)에 따라 활성화 시 밟으면 큰 벌점(-200)을 받고 즉시 실패 처리되어 리셋됩니다.
   * **보너스 (Bonus - 파란 별)**: 획득 시 보상(+150)을 받으며, 중복 획득이 방지되어 에피소드당 단 한 번만 획득 가능합니다.
5. **학습 로그 파일 기록 (`rl_training.log`)**:
   * 에피소드가 끝나거나 최적화 테스트가 진행될 때마다 각 스텝 수, 성공 여부, 파라미터(알고리즘, 학습률, 감가율, 에스필론, 맵 크기)를 CSV 형식으로 실시간 누적 기록합니다.
6. **실시간 시각화 메트릭 및 학습 곡선**:
   * 에피소드 성공 횟수, 최단 경로 기록, 탐험률 변화 메트릭을 실시간 표시합니다.
   * 사이드바 하단에 최근 100회의 에피소드별 소요 스텝 수 변동 추이를 꺾은선 그래프로 시각화합니다.
7. **모델 세이브 및 로드**:
   * `tkinter` 파일 다이얼로그를 통해 현재 설계된 미로와 학습된 Q-Table을 `.npz` 압축 형식으로 컴퓨터에 저장하고 불러옵니다.

## 📁 프로젝트 구조 (`src/` 디렉토리)

코드는 역할별로 모듈화되어 설계되어 가독성과 유지보수성이 뛰어납니다.
```
src/
├── main.py             # 애플리케이션 시작 진입점
├── rl/
│   ├── __init__.py
│   ├── agent.py        # 3가지 RL 알고리즘 및 Q-Table 상태 제어
│   └── environment.py  # 보상, 상태전이 및 타일 정보 정의
├── ui/
│   ├── __init__.py
│   ├── gui.py          # Pygame GUI 렌더링, 제어 패널 이벤트 루프
│   └── widgets.py      # 재사용 가능한 UI 컴포넌트 (Button, Slider)
└── utils/
    ├── __init__.py
    └── generator.py    # DFS 미로 무작위 생성 유틸리티
```

## 🚀 시작하기

### 실행 환경 요구사항
* **Python 3.14.3 이상**
* Pygame-CE, NumPy, Matplotlib, Seaborn 등 (의존성 파일 참조)

### 실행 방법
프로젝트 폴더 내에 배치 파일이 내장되어 있어 가상환경 생성 및 패키지 설치부터 구동까지 원클릭으로 동작합니다.

```powershell
# Windows 환경
.\run.bat
```

또는 직접 가상환경 활성화 후 명령어로 실행할 수 있습니다:
```powershell
# 가상환경 활성화
.\miro_env\Scripts\activate.bat

# 패키지 설치 확인
pip install -r requirements.txt

# 시뮬레이터 구동
python src/main.py
```
